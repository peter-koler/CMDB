## 🚀 启动说明

### 环境要求
- Node.js 18+
- Python 3.11+

### 安装依赖
```bash
# 后端
cd backend
pip install -r requirements.txt

# 前端
cd frontend
npm install
```

### 启动服务
```bash
# 后端（端口5000）
cd backend
python run.py

# 前端（端口3000）
cd frontend
npm run dev
```

### 访问地址
- 前端：http://localhost:3000
- 后端API：http://localhost:5000
- 默认账号：admin / admin
